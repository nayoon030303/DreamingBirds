## 개발 환경
언어 : Html/css, JS, Node j.s<br>
툴 : VS code<br>
DB : Mongo DB

## 기능
1. 캠으로 모습을 인식해서 실제 공부시간을 측정
2. 졸고있는 모습, 휴대폰 하고있는 모습 등등 공부 하지 않는 경우 경고 알림
3. 체크리스트
4. 일간, 주간, 월간으로 통계
